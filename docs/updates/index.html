  • Begone! Google fonts (September 29, 2022)

    Okay, that's a far more dramatic title than this update really is. Long story short: I live in Europe, and German court has ruled that using Google Font's API to serve fonts (as opposed to hosting locally) breaks GDPR. I will be using Bunny Fonts instead, which claims to be a "privacy-first web font platform designed to put privacy back into the internet".
